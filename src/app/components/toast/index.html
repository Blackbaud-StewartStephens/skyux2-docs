The toast module displays messages over the page's content.

The SkyToastService launches toast messages and provides a common look and feel.

Opens a message toast and displays the provided message. The message parameter is of type string and determines the message to be displayed. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. This method returns a SkyToastInstance instance. Opens a custom toast and renders the provided component in a toast. The component parameter determines the component to be rendered. The optional config parameter is of type SkyToastConfig and adds additional configuration options to the toast. The optional providers parameter is of type StaticProvider[] and specifies an array of custom providers to pass to the component's constructor. This method returns a SkyToastInstance instance. Closes all open toasts at once.
